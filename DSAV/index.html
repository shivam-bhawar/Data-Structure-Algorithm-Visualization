Linear Search

Linear Search Algorithm:

Linear Search, also called as 'Orderly Search' or 'Sequential Search', is a very simple Search Algorithm. In this type of Search, a Sequential Search is made over all items one by one. Every item is checked and if a match is found then that particular item is returned, otherwise the Search continues till the end of the Data Collection.

  • Worst Case Complexity: O(n)
  • Average Case Complexity: O(n)
  • Best Case Complexity: O(1)
